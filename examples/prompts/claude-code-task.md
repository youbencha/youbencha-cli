# Claude Code Task: Code Review

You are performing a comprehensive code review of this repository.

## Objectives

1. **Code Quality Analysis**
   - Review code structure and organization
   - Check for consistent naming conventions
   - Identify code duplication opportunities
   - Evaluate function and method complexity

2. **Security Review**
   - Identify potential security vulnerabilities
   - Check for sensitive data exposure
   - Review authentication and authorization patterns
   - Verify input validation and sanitization

3. **Performance Assessment**
   - Identify performance bottlenecks
   - Check for inefficient algorithms
   - Review database query patterns
   - Evaluate caching opportunities

4. **Best Practices**
   - Verify error handling patterns
   - Check for proper logging
   - Review test coverage
   - Evaluate documentation completeness

## Output Format

Provide your findings in a structured format:

```markdown
## Summary
[Brief overview of findings]

## Critical Issues
[List of high-priority issues that need immediate attention]

## Recommendations
[Actionable suggestions for improvement]

## Positive Observations
[Things that are done well]
```

Focus on actionable feedback that can improve code quality.
