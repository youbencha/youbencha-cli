# project-namer

A CLI tool that generates creative project names.

## Installation

### Prerequisites

- Node.js 18.0.0 or higher

### Install

```bash
npm install project-namer
```

### Quick Start

Run the CLI tool to generate a project name:

```bash
npx project-namer
```

Or add it to your npm scripts in `package.json`:

```json
{
  "scripts": {
    "generate-name": "project-namer"
  }
}
```

Then run:

```bash
npm run generate-name
```
